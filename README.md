# Old_DungeonLegend

自作ゲーム「ダンジョンレジェンド」のソースコードです。

## 各ファイルの説明

### Player-HPCheck
プレイヤーのHPを管理します。

### Player-HPText
プレイヤーの現在HPを表示します。

### Player-Player
プレイヤーの状態を持ちます。

### Player-Respawn
プレイヤーのHPが0になったとき、チェックポイントで復活します。

### Block,BlueBlock,RedBrock
Blockは爆弾で破壊出来るブロック、BrueBlockはスイッチをオンにしたときに出現し、RedBlockはスイッチをオフにしたときに出現します。
